# YYR 绘图工具集

一个精简的 AutoCAD LISP 工具集，包含 Brace 支撑架生成、绘制直线、绘制圆形功能。

## 🚀 快速开始

### 最简单的使用方法

```
1. 打开 AutoCAD
2. 将 YYR.lsp 拖放到 AutoCAD 窗口
3. 命令行输入：YYR
4. 开始使用！
```

## 📋 功能清单

| 功能 | 说明 | 推荐度 |
|------|------|--------|
| **Brace** | 选择直线生成两个矩形 | ⭐⭐⭐⭐⭐ |
| 绘制直线 | 输入坐标画线 | ⭐⭐⭐ |
| 绘制圆形 | 输入圆心半径画圆 | ⭐⭐⭐ |

## 🎯 Brace 功能（核心功能）

### 功能描述
选择一根直线，自动生成两个并排矩形。

### 参数设置
- **默认参数：** 50
- **矩形1 宽度：** 50 × 5 = 250
- **矩形2 宽度：** (50-10) × 5 = 200
- **图层：** 03Brace（自动创建）

### 使用步骤
```
1. 输入 YYR
2. 参数输入：50（默认）
3. 点击"执行"
4. 选择一根直线
5. 完成！
```

### 方式 2：APPLOAD
```

### 方式 3：自动加载
```
复制到：C:\Program Files\Autodesk\AutoCAD 2020\Support\
