## 🚀 使用方法

### 快速启动

```
1. 打开 AutoCAD 2020
2. 导航到：03-CAD - Drawing\ 目录
3. 拖放 YYR.lsp 到 AutoCAD 窗口
4. 命令行输入：YYR
5. 开始使用！
```

### 或者使用 APPLOAD

```
1. 打开 AutoCAD 2020
2. 命令：APPLOAD
3. 浏览到：03-CAD - Drawing\YYR.lsp
4. 点击"加载"
5. 命令：YYR
```
