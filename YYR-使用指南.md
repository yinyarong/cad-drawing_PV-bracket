# YYR 绘图工具集 - 使用指南

## 📋 功能概述

**YYR** 是一个精简的 AutoCAD LISP 工具集，包含 3 个主要功能：
1. **Brace** - 选择直线自动生成两个矩形
2. **绘制直线** - 输入坐标绘制直线
3. **绘制圆形** - 输入圆心和半径绘制圆形

---

## 🚀 快速开始

### 方法 1：命令行加载

```
1. 打开 AutoCAD 2020
2. 命令行输入：APPLOAD
3. 浏览并选择：YYR.lsp
4. 点击"加载"
5. 关闭对话框
6. 命令行输入：YYR
```

### 方法 2：拖放加载（最简单）

```
1. 打开 AutoCAD 2020
2. 将 YYR.lsp 文件拖放到 AutoCAD 窗口中
3. 命令行输入：YYR
```

### 方法 3：自动加载（每次启动自动加载）

```
1. 复制 YYR.lsp 到 AutoCAD 支持路径：
   C:\Program Files\Autodesk\AutoCAD 2020\Support\

2. 创建或编辑 acad.lsp 文件，添加：
   (load "YYR.lsp")

3. 重启 AutoCAD，输入 YYR 即可使用
```

## 🎯 功能详解

### 1. Brace 功能（推荐使用）

**功能说明：**
选择一根直线后，自动生成两个并排的矩形。

**参数说明：**
- 参数值：默认 50（可修改）
- 矩形 1 宽度 = 参数 × 5 = 250
- 矩形 2 宽度 = (参数 - 10) × 5 = 200
- 两个矩形以直线为中心线，在直线上方并排排列
- 图层：自动创建 `03Brace` 图层

**使用步骤：**
```
1. 输入 YYR 命令
2. 在对话框顶部 "Brace - 支撑架生成" 区域：
   - 输入参数值（默认 50）
   - 点击"执行"按钮
3. 在 AutoCAD 中点击选择一根直线
4. 自动生成两个矩形！
```

**示例场景：**
```
参数 = 50
→ 矩形1: 250 × 直线长度
→ 矩形2: 200 × 直线长度
→ 图层: 03Brace（白色）
```

**注意事项：**
- ✅ 参数必须 > 10
- ✅ 只能选择 LINE 对象（不能是多段线 PLINE）
- ✅ 直线必须有长度（不能是点）

---

### 2. 绘制直线功能

**功能说明：**
输入起点和终点坐标，在模型空间创建直线。

**使用步骤：**
```
1. 输入 YYR 命令
2. 切换到"绘制直线"区域
3. 输入坐标：
   - 起点: X=0, Y=0, Z=0
   - 终点: X=100, Y=100, Z=0
4. 选择颜色（默认红色）
5. 点击"绘制直线"按钮
```

**颜色选项：**
- 红色 (1)
- 黄色 (2)
- 绿色 (3)
- 青色 (4)
- 蓝色 (5)
- 品红 (6)
- 白色 (7)

---

### 3. 绘制圆形功能

**功能说明：**
输入圆心坐标和半径，在模型空间创建圆形。

**使用步骤：**
```
1. 输入 YYR 命令
2. 切换到"绘制圆形"区域
3. 输入参数：
   - 圆心: X=0, Y=0, Z=0
   - 半径: 50
4. 选择颜色（默认绿色）
5. 点击"绘制圆形"按钮
```

## 📐 Brace 功能技术说明

### 矩形计算逻辑

**已知：**
- 选中的直线长度为 L
- 直线的起点为 P1，终点为 P2
- 用户输入参数为 P（默认 50）

**计算公式：**
```
矩形1:
- 长度 = L（沿直线方向）
- 宽度 = P × 5
- 位置：以直线为中心线，向上偏移宽度的一半

矩形2:
- 长度 = L（沿直线方向）
- 宽度 = (P - 10) × 5
- 位置：矩形1外侧，紧挨着
```

**坐标计算：**
```
直线方向向量: (dx, dy) = (P2.x - P1.x, P2.y - P1.y)
垂直向量（向上）: (-dy, dx) 单位化
矩形偏移:
  - 矩形1: offset1 = (P × 5) / 2
  - 矩形2: offset2 = (P × 5) + ((P-10) × 5) / 2
```

### 图层说明

**自动创建图层：**
- 图层名：`03Brace`
- 颜色：白色（颜色索引 7）
- 线型：Continuous
- 线宽：默认

## ⚠️ 常见问题

### Q1: 输入 YYR 后无反应？

**检查：**
1. 确认 LISP 文件已加载（APPLOAD 查看）
2. 命令行是否显示"YYR 绘图工具集已加载"
3. 尝试重新加载 LISP 文件

---

### Q2: Brace 点击执行后没反应？

**检查：**
1. 确认选择了 LINE 对象（不是 PLINE）
2. 直线有实际长度
3. 参数值 > 10

---

### Q3: 矩形位置不对？

**说明：**
- 矩形以直线为"中心线"
- 矩形在直线的"上方"（Y 轴正方向）
- 两个矩形并排，矩形 2 在矩形 1 外侧

---

### Q4: 图层创建失败？

**解决：**
- 手动创建 03Brace 图层
- 或者以管理员身份运行 AutoCAD

## 🔄 与 C# 版本对比

| 特性 | C# 版本 | LISP 版本（YYR.lsp） |
|------|---------|---------------------|
| **加载方式** | NETLOAD | APPLOAD 或拖放 |
| **界面** | WPF 现代界面 | DCL 传统对话框 |
| **性能** | 编译执行，快 | 解释执行，够用 |
| **兼容性** | 需要特定 .NET | 所有 AutoCAD 版本 |
| **安装** | 需要编译 | 直接使用 |
| **修改** | 需要 Visual Studio | 文本编辑器即可 |
| **文件大小** | ~50 KB | ~15 KB |

**推荐：**
- ✅ 日常使用：LISP 版本（简单、兼容性好）
- ✅ 复杂项目：C# 版本（功能强大、性能好）

## 💡 高级技巧

### 技巧 1: 快速访问

创建快捷命令别名：
```
在 AutoCAD 中：
1. 命令: ALIASEDIT
2. 添加别名: Y = YYR
3. 保存
4. 以后只需输入 Y 即可
```

### 技巧 2: 批量处理

如果要为多根直线创建矩形：
```
1. 手动调用 Brace 功能
2. 每次选择一根直线
3. 参数保持不变
4. 重复操作
```
